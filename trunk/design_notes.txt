Path management
--------------------------------------------------------------------------------------------

Tipi di path in base alla risorsa, all'azione
	1)Normal resource
		a.Actions with id
			I.hittp://myhost.com/node/view/itemid(int)/page(int)
			II.hittp://myhost.com/cathegory/view/cathegory_name/page(int) 
		b.Actions without id
			I.hittp://myhost.com/node/create/type_name/
			II.hittp://myhost.com/node/create/type_name/cathegory_name
	2)Special resource
		a.Actions with id
			I.hittp://myhost.com/admin/box/create/type_name/
		b.Actions without id
			I.hittp://myhost.com/admin/box/create/type_name/
	3)Automatic partial aliasing : hittp://myhost.com/node/view/item_title(string)_itemid(int)_ppage(int)
	4)Custom aliasing (no pag nr here): hittp://myhost.com/custom_path/custom_title
